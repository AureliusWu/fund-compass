# Changelog

## 6.0.0 - 2026-07-22

- 详情决策接入后端实时估值上下文与用户持仓状态，形成可解释的操作闭环。
- 未持有输出买入、分批定投、观望；已持有输出加仓、持有、减仓、卖出。
- 新增强度、数据状态/时间、相对位置、趋势、操作方式、风险和结论变化条件。
- PWA 对 `/api` 改为 NetworkOnly，浏览器请求使用 `no-store`，避免旧决策静默回退。
- 移除 GitHub Actions 盘中定时信号轮询；估值推送仅保留人工应急，正式任务由 Cloudflare Worker 承担。
- 拆分持仓富集、基金经理富集和策略校准工作流，并强化 CI、Pages 部署与部署后冒烟检查。
